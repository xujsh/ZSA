jQuery警告/确认/提示弹出对话框效果（替换传统JavaScript下的提示框）

基本示例

警告提示框（Alert） 
jAlert('这是一个可自定义样式的警告对话框', '警告对话框');

确认对话框（Confirm） 
jConfirm('你确定这么做吗?', '确认对话框', function(r) {
    jAlert('是否确定 ' + r, '确定内容');
});

带输入的提示框（Prompt） 
jPrompt('输入字符:', '这里是默认值', '带输入框的提示框', function(r) {
    if( r ) alert('你输入的是 ' + r);
});

高级示例

带HTML输出效果的对话框 
jAlert('你可以使用HTML标签，比如 加粗, 斜体，和 下划线!');
